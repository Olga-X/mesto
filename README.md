# Проект: Место

### Обзор
Четвертая проектная работа по курсу Яндекс Практикум. 
В ней использовались:
1. БЭМ структура
2. Flex и Grid для фотогалереи
3. Созданы с помощью JavaScript - всплывающее окно (popup) с активной формой ввода информации
